17. API RULES
============================================================

- JSON only
- HTTPS only
- RESTful naming
- JWT for admin endpoints
- No permanent delete
- Soft delete for citizen data
- Versioned APIs
- Validation before processing
- Every response follows a standard envelope
- All timestamps in UTC
- UUID used for all primary keys
