1. Introduction
Purpose

This document defines the complete User Experience, User Interface, Design System, Interaction Patterns, Visual Identity, Accessibility Standards and Frontend Architecture guidelines for the AI-Powered Citizen Question Intelligence Platform.

It acts as the single source of truth for:

UI Designers
UX Designers
Frontend Developers
AI-assisted Code Generation Tools
QA Engineers
Product Managers
Objective

Design a platform that enables any citizen of West Bengal to easily submit governance-related questions while providing government officials with an AI-powered analytics dashboard that transforms thousands of questions into meaningful insights.

Saffron Usage Rule

Saffron should appear throughout the interface as a subtle, consistent accent rather than as a dominant background color. It should naturally guide attention to important actions and information without overwhelming the user.

Color Ratios

Warm Off White → 75%

Paper Warm → 10%

White Cards → 8%

Typography → 5%

Saffron Accent → 2%

This balance keeps the interface calm while ensuring every page carries a recognizable saffron identity.

Saffron Design Language

The saffron accent should be visible in nearly every screen through small but intentional details:

Active navigation underline
Primary call-to-action buttons
Wizard step indicators
Focus outlines
Hyperlinks
Icons for important actions
AI insight badges
Left accent border on AI cards
Hover states
Selected chips
Chart highlight series
Notification badges
Report cover strip
Pagination active state

Avoid using saffron for large page backgrounds, dense text blocks, or oversized decorative sections.
