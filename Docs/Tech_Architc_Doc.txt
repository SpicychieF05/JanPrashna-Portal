1. System Overview
Purpose

JanPrashna Portal is an AI-powered citizen intelligence platform that enables residents of West Bengal to submit governance-related questions.

Instead of acting as a grievance portal, the platform aggregates, analyzes, clusters, and summarizes citizen questions into structured reports for government officials.

The platform is composed of three independent applications:

Citizen Portal
Admin Dashboard
Backend API

These applications share reusable libraries through a centralized monorepo architecture.

2. Architecture Principles
2.1 Scalability

The system should support millions of submissions without requiring architectural redesign.

Scaling should be horizontal wherever possible.

2.2 Separation of Concerns

Citizen website

↓

Backend API

↓

AI Workers

↓

Database

↓

Admin Dashboard

Every layer has a single responsibility.

2.3 AI First

AI processing should never block user requests.

Every submission enters an asynchronous processing queue.

2.4 Shared Code

The citizen portal and admin dashboard should never duplicate:

UI Components
Types
Validation
Constants
Utilities

Everything reusable lives inside shared packages.

2.5 Security by Design

Every layer should follow Zero Trust Architecture.

No service trusts another service without authentication.
